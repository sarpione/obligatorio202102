Partiendo del repositorio https://github.com/emverdes/obligatorio202102.git. 

Se edita el playbook para instalar LAMP en 2 servidores (uno Centos y otro Ubuntu)

Los cambios que se hacen son:

- Se elimina el rol "Common" y se agrega el rol "PHP"
- Se edita el rol DB de forma que ya no instale MySQL y ahora instale MariaDB
- Se edita el rol Web para que instale Apache en sus versiones para Centos y Ubuntu
- En el rol PHP se configura la instalacion del modulo PHP para las versiones de Centos y Ubuntu.
- Se configura que al final del playbook se habilite Selinux en el servidor Centos.
